**基于webpack3.5.5构建react单页面**

----------
项目地址：<https://github.com/GoFighting/webpack-react>

技术栈
----------
react + redux + webpack3 + react-router + ES6/7 + immutable

运行项目
----------
    git clone https://github.com/GoFighting/webpack-react
    cd webpack-react
    yarn install
    npm run dll（公共模块分离，提高打包速度）
    npm start 
    npm run build（打包）
